descrição detalhada

Lúdico e educativo, este lindo kit de garfo e faca é perfeito para as crianças que estão prontas para começar a comer sozinhas. Seu design inovador alia beleza, conforto e segurança, deixando a refeição dos pequenos muito mais prazerosa.

Os cabos possuem formato curvo para facilitar o manuseio e detalhes antiderrapantes que evitam que os talheres caiam dentro do prato ou saiam rolando pela mesa. As pontas do garfo são arrendondadas o suficiente para garantir maior segurança, mas ainda permitindo que a comida seja espetada eficientemente. A colher possui curvatura perfeita para segurar melhor o alimento, minimizando as frustrações de quem está aprendendo a se alimentar sozinho.

Recomendado para crianças a partir de 1 ano de idade.
